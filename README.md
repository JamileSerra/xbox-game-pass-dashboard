# 🎮 Dashboard de Receita – Xbox Game Pass

## 📊 Visão Geral do Projeto
Este projeto consiste na criação de um dashboard em Excel para análise de vendas de assinaturas do Xbox Game Pass.

O objetivo é analisar o faturamento, entender o comportamento dos clientes e avaliar o impacto dos diferentes planos e add-ons na receita total.

---

## 🎯 Perguntas de Negócio

O dashboard responde às seguintes perguntas:

1. Qual o faturamento total de vendas de planos anuais?
2. Qual o faturamento de planos anuais separado por auto renovação e não auto renovação?
3. Qual o faturamento gerado pelo EA Play Season Pass?
4. Qual o faturamento gerado pelo Minecraft Season Pass?

---

## 📁 Base de Dados

A base contém informações como:

- ID do cliente
- Nome
- Plano (Core, Standard, Ultimate)
- Tipo de assinatura (Monthly, Quarterly, Annual)
- Status de auto renovação
- Preço da assinatura
- Add-ons (EA Play e Minecraft Season Pass)
- Valor de cupons
- Valor total da compra

---

## 📊 Dashboard

O dashboard apresenta:

- 💰 Receita Total
- 🎮 Receita com EA Play
- 🟩 Receita com Minecraft Season Pass
- 🔁 Receita anual por tipo de renovação
- 📦 Receita anual por plano

## ▶️ Como utilizar

1. Baixe o arquivo Excel deste repositório
2. Abra no Microsoft Excel
3. Acesse a aba **Dashboard**
4. Utilize os filtros para interagir com os dados

---

## 📌 Principais Insights

- O plano Ultimate gera o maior faturamento anual
- Uma parte relevante da receita vem de assinaturas sem auto renovação
- Os add-ons contribuem significativamente para o faturamento total

---

## ⚠️ Observação sobre os dados

Foram realizados pequenos ajustes na base de dados original fornecida pela DIO, com o objetivo de padronização e melhoria da análise.

Além disso, o valor do plano anual foi mantido constante ao longo da base ("price freezing"), com o objetivo de garantir consistência na análise de faturamento e evitar distorções nos resultados.
